Clothing slideshow example:

  • You may use the buttons below to edit the slideshow.

Buttons to edit the clothing slideshow:
  • These buttons utilize the orderForward() and orderBackward() functions to change the order of the bottoms(pants and skirts) slideshow below.
    note: the order is not relative to other layers so if you order a layer forward multiple times you must reorder it back multiple times to get the layer back to its original order

    For example, if you were to want the long pants to appear above the shoes, press the 'bottoms forward' button and to move it back, press the 'bottoms backwards' button.
  • This button utilizes the remove slide function to remove the white shoes from the shoe slideshow
