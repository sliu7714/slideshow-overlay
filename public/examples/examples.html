You can create a simple slideshow with only the background and no foreground slideshows.

    const bgSlides = []
    // create the background slides
    for (let i = 0; i < imageURLs.length; i ++){
        bgSlides.push(new BackgroundSlide(imageURLs[i], 'nature photo', `bg_${i}`))
    }

    // add slides to background slideshow
    const bgSlideshow = new BackgroundSlideshow('bg-slideshow', bgSlides, {height: "500px"})

    // add only the background slideshow to DOM 
    const simpleExampleSlideshowContainer = new SlideshowContainer('simple-ex-slideshow-container', 'simple-example-slideshow-div', bgSlideshow)

Text Slides and Adding a Foreground Slideshow

You can also create add additional foreground slideshows that contain either images or text. 
                            
    const greetingText = ["Hello", "Bonjour", "Ciao", "Hola"]
    const textSlides = []
    // create the foreground text slides
    for (let i = 0; i < greetingText.length; i ++){
        textSlides.push(new TextSlide(greetingText[i], 30, `greeting${i}`))
    }
    // add slides to foreground slideshow
    const textSlideshow = new ForegroundSlideshow('text-greeting-slideshow', textSlides, {height: "100px", width: "390px", marginTop: "120px"} )
    ...
    // add slideshow to the slideshow container
    simpleExampleSlideshowContainer.addForegroundSlideshow(textSlideshow)
    ...
    // remove slideshow from the slideshow container
    simpleExampleSlideshowContainer.removeForegroundSlideshow(textSlideshow)
  • Press this button to add/remove a foreground text slideshow

Auto-Scroll

There is the option to automatically change the slides for each slideshow. 
                            
    ...
    // set auto scroll to change slide every 2 seconds
    bgSlideshow.addAutoScroll(2000)
    ...
    // turn auto scroll off
    bgSlideshow.removeAutoScroll()
  • Pressing this 'start auto-scroll' button will change the background slideshow bgSlideshow to change slides every 2 seconds. Then pressing 'stop auto-scroll' will stop the automatic scrolling.

Clothing Slideshow

This example shows more styling and options for slideshow objects 
                
    // BACKGROUND SLIDESHOW SETUP
    const clothesBgSlides = []
    const numClothesBgSlides = 7
    // make background slide objects
    for (let i = 1; i <= numClothesBgSlides; i++){
        clothesBgSlides.push(new BackgroundSlide(`../images/clothes/bg/bg${i}.png`, 'person outline', `clothing-bg${i}`))
    }
    // make backgroundSlideshow
    const clothesBgSlideshow = new BackgroundSlideshow('clothes-bg-slideshow', clothesBgSlides, {height: "500px"})
    
    // FOREGROUND SLIDESHOW SETUP
    const clothesTopsSlides = []
    const numTops = 8
    for (let i = 1; i <= numTops; i++){
        clothesTopsSlides.push(new ForegroundSlide(`../images/clothes/tops/top${i}.png`, 'shirt', `clothing-top${i}`))
    }
    const clothesTopsSlideshow = new ForegroundSlideshow('clothes-tops-slideshow', clothesTopsSlides,{height: "145px", marginTop: "100px", marginLeft: "40px" })
    
    const clothesBottomsSlides = []
    const numBottoms = 5
    for (let i = 1; i <= numBottoms; i++){
        clothesBottomsSlides.push(new ForegroundSlide(`../images/clothes/bottoms/bottom${i}.png`, 'clothing bottom', `clothing-bottom${i}`))
    }
    const clothesBottomsSlideshow = new ForegroundSlideshow('clothes-bottoms-slideshow', clothesBottomsSlides, {height: "245px", marginTop: "200px", marginLeft: "38px"})
    
    const clothesShoesSlides = []
    const numShoes = 3
    for (let i = 1; i <= numShoes; i++){
        clothesShoesSlides.push(new ForegroundSlide(`../images/clothes/shoes/shoe${i}.png`, 'shoe', `clothing-shoe${i}`))
    }
    const clothesShoesSlideshow = new ForegroundSlideshow('clothes-shoes-slideshow', clothesShoesSlides, {height: "150px", marginTop: "350px", marginLeft: "40px"})
    
    // Add foreground and background slideshows to DOM
    const clothesSlideshowContainer = new SlideshowContainer('clothes-slideshow-container', 'clothing-slideshow-div', clothesBgSlideshow, clothesTopsSlideshow)
    clothesSlideshowContainer.addForegroundSlideshow(clothesBottomsSlideshow)
    clothesSlideshowContainer.addForegroundSlideshow(clothesShoesSlideshow)
  • You may use the outer arrows to change the model's skin tone
  • The arrows in the 3 inner foreground slideshows can be used to change the shirt,bottom, and shoes on the model

Changing slides

On top of being able to navigate to the next and previous slide, you may also navigate to a random slide in the slideshow.
Use the "random shirt" and "random outfit" buttons below to try it out: 
                            
    ...
    // change a single slideshow to a random slide
    clothesTopsSlideshow.randomSlide()
    ...
    // change all the slideshows in a slideshow container to a random slide
    clothesSlideshowContainer.randomSlides()

Changing layer orders

There is the option to re-order the appearance of the ForegroundSlideshows in a SlideshowContainer so that one slideshow appears to be in front of another.
Use the buttons below to re-order the pants/skirt slideshow to appear below/above the other clothing slides 
                            
    ...
    // move skirt/pants slideshow forward one layer
    clothesBottomsSlideshow.orderForward()
    ...
    // move skirt/pants slideshow backwards one layer
    clothesBottomsSlideshow.orderBackward()

    // NOTE: the order is not relative to other layers so if you move a slideshow a layer
    //       forward multiple times you must reorder the slideshow back multiple times
    //       to get the slideshow back to its original order

Adding/Removing slides

Slides may be added/removed after creating a slideshow.
Use the buttons below to try adding and removing a slide

Gift Box Slideshow

This gift box example showcases more of the arrow styling options
  • The outer black arrows will change the gift box's colour.
  • The orange-red arrows will change the colour of the crinkle packing paper.
  • Hovering your mouse over the items inside the box will reveal blue, pink and green arrows that
    will change the items (bear, handkerchief, and cup) included in the gift box.

Code with arrow styling options emphasized: 
                        
    // see documentation page for more details about options

    ... create gift box slides ...

    // create gift box slideshow
    const giftBoxSlideshow = new BackgroundSlideshow(
        id='gift-boxes-slideshow',
        slides=giftBoxSlides,
        // not having any arrow styling options defaults to a black variant 1 arrow style
        options={
            height: "500px"
        } 
    )
    
    ... create gift paper slides ...

    // create gift paper slideshow
    const giftPaperSlideshow = new ForegroundSlideshow(
        id='gift-paper-slideshow', 
        slides=giftPaperSlides, 
        // the arrowVariant and arrowColor options set the gift paper slideshow to have round orange arrows 
        options={
            height: "373px", 
            marginLeft: "76px",
            arrowVariant: 2,      // the variant style for the arrows
            arrowColor: "#ff6347" // the color of the arrows
        }
    )
    
    ... create bear slides ...

    // create teddy bear slideshow
    const giftBearSlideshow = new ForegroundSlideshow(
        id='gift-bear-slideshow', 
        slides=giftBearSlides, 
        // the arrowVariant and arrowColor options set the bear slideshow
        // to have triangle blue arrows that only appear when hovering on the bear slideshow
        options={
            height: "180px", 
            marginLeft: "190px",
            onlyShowArrowOnHover: true , // option to only show the arrow when hovering over the slideshow
            arrowVariant: 3,             // the variant style for the arrows   
            arrowColor: "deepskyblue"    // the color of the arrows
        }
    )
