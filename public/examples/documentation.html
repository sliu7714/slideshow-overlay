Overlay Slideshow Examples Documentation Github

Getting Started

To use the library, include these scripts: 
                < link rel="stylesheet" href="'<...path to overlaySlideshowStyles.css>">
                    < script src="<...path to overlaySlideshow.js>"> </script>

Creating Slides

Before creating a slideshow, you must create slides first.
There are 3 types of slides,
  • BackgroundSlides contain images and can be added to BackgroundSlideshows
  • ForegroundSlides contain images and can be added to ForegroundSlideshows
  • TextSlides contain text and can be added to ForegroundSlideshows
Here is an example of creating a simple BackgroundSlide instance, 
                 const backgroundSlide = new BackgroundSlide('<alt text for image>', '<...path to image>', 'background-slide-element-id' )

Creating Slideshows

After creating the slide instances, you can create slideshows (BackgroundSlideshow or ForegroundSlideshow)
Here is an example of creating a simple BackgroundSlideshow instance 
                 const backgroundSlideshow = new BackgroundSlideshow('background-slideshow-element-id', [...list of BackgroundSlide instances], {...options})

Note:

each slideshow must contain at least one slide and slide instances cannot be used in more than one slideshow.

Adding to SlideshowContainers

Finally, after creating Slideshow instances, you may create a SlideshowContainer to add the slideshows to the DOM 
                 
                    const slideshowContainer = new SlideshowContainer('slideshow-container-element-id', 'id of element to append this SlideshowContainer to', < BackgroundSlideshow instance >, < ForegroundSlideshow instance >(optional))
SlideshowContainers can only contain one BackgroundSlideshow, but they may contain zero or more ForegroundSlideshows

Constructors

BackgroundSlide(alt, src, id)

Create a new background image slide to be used in a BackgroundSlideshow.
Does not add this slide to the DOM
Parameters
alt the alt text of the slide image
src the link to the image to use for this slide
id the id to use for this slide, this id will also be used for the html element corresponding to this slide

ForegroundSlide(alt, src, id)

Create a new foreground image slide to be used in a ForegroundSlideshow.
Does not add this slide to the DOM
Parameters
alt the alt text of the slide image
src the link to the image to use for this slide
id the id to use for this slide, this id will also be used for the html element corresponding to this slide

TextSlide(text, fontSize, id)

Create a new foreground text slide to be used in a ForegroundSlideshow.
Does not add this slide to the DOM
Parameters
text the text to use inside the slide
fontSize the size of the font to use in this slide
id the id to use for this slide, this id will also be used for the html element corresponding to this slide

ForegroundSlideshow(id, slides, options)

Create a new foreground slideshow that may contain either TextSlide or ForegroundSlide instances
This slideshow must be added to a SlideshowContainer to be added to the DOM
Parameters
id the id to use for this slideshow, this id will also be used for the html element corresponding to this slideshow
slides a list of TextSlide or ForegroundSlide instances to be added to this slideshow
this list cannot be empty, every slideshow must contain at least one slide
options options to style/manipulate this slideshow, see the options section below for more information

BackgroundSlideshow(id, slides, options)

Create a new background slideshow that contains BackgroundSlide instances
This slideshow must be added to a SlideshowContainer to be added to the DOM
Parameters
id the id to use for this slideshow, this id will also be used for the html element corresponding to this slideshow
slides a list of BackgroundSlide instances to be added to this slideshow
this list cannot be empty, every slideshow must contain at least one slide and the same slide instance cannot be a part of multiple slideshows
options options to style/manipulate this slideshow, see the options section below for more information

SlideshowContainer(id, containerID, backgroundSlideshow, foregroundSlideshow)

Create container to hold one BackgroundSlideshow and zero or more ForegroundSlideshows.
This will add the slideshows to the DOM
Parameters
id the id to use for this slideshow container, this id will also be used for the html element corresponding to this slideshow container
containerID the id of the html element to append this new slideshowContainer to
backgroundSlideshow The BackgroundSlideshow instance to add to this slideshow container
backgroundSlideshow (OPTIONAL) A ForegroundSlideshow instance to add to this slideshow container
Note that slideshow instances (ForegroundSlideshow or BackgroundSlideshow) cannot be added to more than one SlideshowContainer

Methods

Methods for both ForegroundSlideshow and BackgroundSlideshow

ForegroundSlideshow and BackgroundSlideshow both inherit from the same parent class so they share many methods.

.addSlide(slide)

Add a single slide instance to the end of a slideshow

.addSlides(slidesList)

Add a list of slide instances to the end of a slideshow

.removeSlide(slide)

Remove a slide from a slideshow, if the slide is not part of the slideshow or is the last slide in the slideshow, an error is thrown
Every slideshow must contain at least one slide

.nextSlide()

Move to the next slide in the slideshow, or the first slide if the current slide is the last

.prevSlide()

Move to the next slide in the slideshow, or the first slide if the current slide is the last

.changeSlide(index)

Move to a i-th slide in the slideshow corresponding to the parameter 'index'throws an error if the index is out of bounds

.randomSlide()

Move to a random slide in the slideshow

.addAutoScroll(milliseconds)

Set a interval to automatically change the slideshow to the next slide every x milliseconds corresponding to the 'milliseconds' parameter

.removeAutoScroll()

Stop the interval that automatically changes the slides every x milliseconds.

.hideArrows()

Hide the arrows on either side of the slideshow from appearing.

.showArrows()

Show the arrows on either side of the slideshow.

Methods for only ForegroundSlideshow

.orderForward()

Move this slideshow forward one layer (changes the z-index)

.orderBackward()

Move this slideshow backwards one layer (changes the z-index)

SlideshowContainer Methods

.addForegroundSlideshow(foregroundSlideshow)

Add a single ForegroundSlideshow instance to a SlideshowContainer

.removeForegroundSlideshow(foregroundSlideshow)

Remove a ForegroundSlideshow instance from a SlideshowContainer, if the foregroundSlideshow is not a part of the SlideshowContainer, an error is thrown

.randomSlides()

Change all slideshows in this slideshow container to a random slide
